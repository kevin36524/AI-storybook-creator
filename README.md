## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create .env.local or and copy stuff from .env.local_ref
   `cp .evn.local_ref .env.local`
3. Update the .env.local with your API KEYS.
4. Run the app:
   `npm run dev`
